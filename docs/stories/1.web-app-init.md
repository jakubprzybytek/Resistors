# Web App Initialization

The current TypeScript project should be transformed into a real web application built with React and styled using SCSS/Sass, then deployed to AWS with the latest SST version as a static site.

- Replace the current TypeScript-only setup with a React web application.
- Use SCSS or Sass for styling.
- Deploy the app to AWS using the latest SST library.
- Configure SST so the stack is centered around a StaticSite construct that builds TypeScript source into static assets.
- Provide a textarea where users can define available resistor values (comma-separated or newline-separated).
- Provide a second text input for the target resistor value.
- Add a button that calculates and shows a resistor configuration that matches the target value.
